# Etch A Sketch

A simple Etch-a-Sketch web app to practice **DOM manipulation** and **event handling** with JavaScript. Draw directly on a grid by hovering your mouse over the cells, and reset the canvas with a click.

## Features

- Dynamic grid creation
- Mouse hover drawing
- Clear button
- Responsive and interactive

## How to Use

1. [Click here to open the page](https://kristijanxo.github.io/odin-etch-a-sketch/)
2. Hover over the grid to draw.
3. Click the "Clear" button to clear the drawing.
4. Set a custom grid resolution by clicking the "Change grid" button

## Learning Outcomes

- Creating and modifying DOM elements dynamically
- Handling DOM events (`mouseover`, `click`)
- Working with CSS classes programmatically
